Help: Wfdownloads

Fresh Install

These instructions are for a fresh install.

If you want to update an older Wfdownloads please select 'Update'.
If you want to import from Xoops myDownloads please select 'Import'.

We bLY recommends you take a full backup before INSTALLING or UPDATING any module. .

Fresh installation of Wfdownloads

Upload the module to your website

Upload the 'wfdownloads' folder to {xoops-rootdirectory}/modules folder

Change and verify folder permissions

CHMOD the following folders to 777:
wfdownloads/images
wfdownloads/images/category
wfdownloads/images/category/thumbs
wfdownloads/images/screenshots
wfdownloads/images/screenshots/thumbs

Also make sure that the {xoops-rootdirectory}/uploads folder is set to 777.

Install the module

Login as administrator and enter Xoops Administration page.
Select System --> modules and install wfdownloads from there

Configure the module

Most important steps now will be to setup the group permissions for the module and its blocks via System --> groups
and than within wfdownloads administration Mimetype management to decide which files are allowed to be uploaded
by users and administrators.
