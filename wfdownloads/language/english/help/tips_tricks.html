Help: Wfdownloads

Upload files

How set maximum upload file size limit?

Maximum upload file size limit is the mimimum value between:

  • Max upload size permitted: upload_max_filesize directive in php.ini
  • Max post size permitted: post_max_size directive in php.ini
  • Memory limit: memory_limit directive in php.ini
  • Module preferences: [upload files] Max file size (bytes)
How can I upload files bigger than maximum upload file size limit?

If you wish to upload a bigger file you should:

  1. use an ftp client to upload file on Module preferences > [upload files] Batch directory directory
  2. just uploaded files are listed in Administration > WF-Downloads > Downloads panel > Batch files table
  3. possible actions are:
    • add: to add file to downloads and edit propierties (this action will remove file from batch directory)
    • delete: to delete file from batch directory

Download files

Are there some limits in download file size?

No, to avoid errors by exceeding the memory_limit setting this module splits big files and serves them chunk by chunk.
